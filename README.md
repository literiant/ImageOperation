# ImageOperation

这是一个面向初学者(也就是现时的本人)的练习仓库，用于探索、了解并手动解析 BMP（位图）图像格式，并通过简单的 C 语言文件操作代码对 BMP 文件进行读取与修改。

项目目标
- 介绍 BMP 文件的基本结构（文件头、信息头、调色板、像素数据）。
- 提供简单的 C 语言示例，演示如何打开 BMP 文件、读取头部信息、修改像素数据并写回文件。
- 帮助初学者理解二进制文件格式、小端字节序和行对齐（padding）等概念。

仓库结构
- `Note/`：包含文档笔记，如 `note.md`（BMP 文件头查看方法详解）。
- `Test/`：包含练习代码和示例文件（如 `test.c`）。

快速开始
1. 阅读 `Note/note.md`，了解 BMP 的文件头和信息头字段含义。 
2. 在 `Test/` 中查看和编译示例 C 文件
3. 运行示例前请先备份要操作的 BMP 文件。示例程序通常会打开 BMP，读取头部信息并对像素进行简单修改（例如翻转颜色或修改透明度），然后保存为新文件。

学习提示
- 注意 BMP 中像素的 BGR 存储顺序（不是 RGB）。
- 每行像素数据按 4 字节对齐，可能包含填充字节（padding）。
- 使用十六进制编辑器（或在 C 中读取并打印头部字段）来验证解析结果。
